# 🎲 3D Board Game (OpenGL)

> **컴퓨터 그래픽스** 수업 팀 프로젝트 — OpenGL/GLUT 기반 3D 2인용 보드 게임

<br>

## 📖 프로젝트 소개

OpenGL과 GLUT를 이용해 구현한 **3D 2인용 보드 게임**입니다.  
두 플레이어(Red · Blue)가 주사위를 굴려 번갈아 가며 말을 이동시키고, 7×7 격자 보드를 뱀처럼 순회하여 **먼저 결승 칸에 도달하는 쪽이 승리**합니다.  
보드 위에는 **포탈(Portal)** 이 배치되어 있어 특정 칸에 도달하면 다른 칸으로 순간이동합니다.

| 항목 | 내용 |
|------|------|
| 개발 인원 | 2인 (서영우 · 박동희) |
| 언어 | C++ |
| 그래픽 라이브러리 | OpenGL (GLUT / GLU) |
| 플레이어 수 | 2인 (Red vs Blue) |

## 🎮 게임 규칙

1. **게임 시작** — `R` 키를 눌러 주사위를 굴리면 짝수/홀수 결과에 따라 **선공(Red/Blue)** 이 결정됩니다.
2. **턴 진행** — 자기 차례에 `R` 키를 눌러 주사위를 굴리고, 결과만큼 말이 자동으로 이동합니다.
3. **이동 경로** — 보드는 7×7 격자이며 말은 *뱀 모양(Snake-path)* 으로 7줄을 순차 이동합니다.
4. **포탈** — 특정 칸(포탈 입구)에 도달하면 연결된 포탈 출구로 순간이동합니다.
5. **잡아먹기** — 이동 후 두 말의 위치가 겹치면 상대 말이 시작점으로 되돌아갑니다.
6. **승리 조건** — 마지막 행의 끝 칸(`y=12, x>12`)에 도달하면 해당 플레이어가 승리합니다.

## 🗺️ 포탈 목록

| 포탈 번호 | 입구 (x, y) | 출구 (x, y) |
|:---------:|:-----------:|:-----------:|
| 1 / 2 | (8, 2) | (4, 6) |
| 3 / 4 | (2, 2) | (6, 10) |

> 포탈에 진입하면 입구·출구 모두 파이프가 시각적으로 연결됩니다.

## 🕹️ 조작 방법

| 입력 | 동작 |
|------|------|
| `R` / `r` | 주사위 굴리기 (선공 결정 or 이동) |
| `1` | 카메라 뷰 1 (가까운 시점) |
| `2` | 카메라 뷰 2 (중간 시점) |
| `3` | 카메라 뷰 3 (전체 시점) |
| 마우스 좌클릭 드래그 | 자유 시점 회전 |
| 우클릭 → Cheat 메뉴 | 주사위 값 강제 지정 (1~6) |

<br>

## ⚙️ 주요 기능 및 구현

### 🎲 주사위 시스템
- 주사위가 50회 회전 후 랜덤으로 멈추며, 회전 축(`X`, `Y`, `Z`)도 무작위 선택
- 멈춘 각도(`T`)와 회전 방향(`SpinCount`)을 조합해 1~6 결과 계산
- **Cheat 메뉴**(우클릭)로 원하는 결과를 강제 지정 가능

### 🏃 플레이어 이동
- 이동 중 매 프레임 `0.05` 단위로 부드럽게 슬라이딩
- 행(Row) 끝에 도달하면 다음 행으로 자동 전환 (P1mark / P2mark 부호로 좌우 방향 제어)

### 🌀 포탈 애니메이션
- 포탈 진입 시 GLU 실린더 + 솔리드 토러스로 3D 파이프 시각화
- 플레이어 크기 애니메이션(`glScalef`) 으로 순간이동 효과 연출

### 💡 OpenGL 조명
- `GL_LIGHT0` 단일 광원 (Ambient + Diffuse + Specular)
- `GL_COLOR_MATERIAL` 로 재질 색상 연동

## 🛠️ 빌드 방법

### 요구 사항
- OpenGL
- GLUT (FreeGLUT 또는 OpenGL Utility Toolkit)
- C++ 컴파일러 (MSVC / GCC / Clang)

### Windows (Visual Studio)
1. 프로젝트에 `boardgame.cpp` 파일 추가
2. 링커 설정에 `opengl32.lib`, `glu32.lib`, `glut32.lib` 추가
3. 빌드 → 실행

### Linux / macOS
```bash
g++ boardgame.cpp -o boardgame -lGL -lGLU -lglut
./boardgame
```
